# Project Scope Statement

## Project Information

**Project Name:** Texas Tech-Themed Portfolio Website
**Project Manager:** John Hazuka
**Project Type:** Two-page portfolio website

## Project Purpose

The purpose of this project is to create a Texas Tech-themed portfolio website that highlights some of the best projects I have completed. The website will give instructors, classmates, recruiters, and potential employers a simple way to view my work, skills, education, and experience.

## Project Description

The project will be a two-page portfolio website created with HTML and CSS. The design will use Texas Tech-inspired colors such as red, black, white, and gray mixed with a rustic cowboy esthetic like tan paper backgrounds animations etc.

The homepage will introduce me and display my featured projects that display my skills the best. The About page will include more information about my education, skills, professional background, and career goals.

## Key Stakeholders

### Project Manager

John Hazuka will plan, design, build, test, and submit the project.

### Course Instructor

The instructor will review the project and determine whether it meets the assignment requirements.

### Website Visitors

Website visitors may include classmates, recruiters, employers, and anyone interested in viewing my work.

## Desired Results

* Create a professional yet fun portfolio website.
* Highlight my strongest completed projects.
* Show my education, experience, and technical skills.
* Make the website easy to navigate.
* Follow UCD principals 
* Use a consistent Texas Tech-inspired design.

## Project Deliverables

The final GitHub repository will include:

* `README.md`
* `index.html`
* `about.html`
* `style.css`

The `docs/` folder will include:

* `scope.md`
* `plan.md`
* `retrospective.md`

## Website Content

### Homepage

The homepage will include:

* My name
* A short introduction
* Featured projects
* Project descriptions
* Technologies used
* Links to GitHub repositories or live projects
* Navigation to the About page

### About Page

The About page will include:

* My education
* My technical skills
* My professional background
* My career interests
* Navigation back to the homepage

## Project Exclusions

The project will not include:

* User accounts
* A database
* An online store
* Incomplete projects
* Official Texas Tech branding or claims of university ownership
